1. 负载均衡

   负载均衡技术，用来在多个计算机（计算机集群）、网络连接、CPU、磁盘驱动器或其他资源中分配负载，以达到最佳化资源使用、最大化吞吐率、最小化响应时间、同时避免过载的目的。

   可以通过 HTTP 重定向、DNS、反向代理、NAT等方式实现。
